== Description ==

WSC Product Title Case is a lightweight plugin that enhances the appearance of your WooCommerce store by converting product titles to title case exclusively on single product pages. This ensures a professional and consistent display of product titles without affecting other parts of your site.

**Key Features**:
* Automatically formats product titles to title case on single product pages.
* Works seamlessly with WooCommerce and most WordPress themes.
* Lightweight and optimized for performance.
* Easy to useâ€”no configuration required.
* Enhances the readability and visual appeal of your product titles.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/product-title-capitalize/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Titles will automatically be updated to title case on single product pages.

== Frequently Asked Questions ==

= Does this plugin work on archive pages? =
No, this plugin only applies title case formatting on single product pages.

= Is it compatible with custom product titles? =
Yes, the plugin dynamically applies title case formatting to all product titles on single product pages.

= Does this plugin work with PHP 8? =
Yes, it has been tested and works with PHP versions 7.2 through 8.3.

= Can I disable the title case formatting for certain products? =
Currently, the plugin applies formatting universally to all single product pages.
